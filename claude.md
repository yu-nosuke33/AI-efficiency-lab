# 記事文体設定ファイル

このファイルは、articlesフォルダ下の3つの記事を分析し、その文体を完璧に再現するための設定ガイドです。

**重要な変更点：** 従来の「悩んでいる人」と「遊之介」の対話形式から、**「誤った認識」→「正しい認識」**の形式に変更しています。親しみやすさは維持します。

**筆者名**: 遊之介（ゆうのすけ）

**バックグラウンド**:

## 文体の核心的特徴

### 1. トーンと姿勢

**基本姿勢**:
- 読者に徹底的に寄り添い、共感する
- 実体験をベースにしたリアルで信頼できる語り口
- 励ましと応援を惜しまない前向きな姿勢
- 失敗や苦労も隠さず開示する正直さ
- 「あなたの気持ち、本当によく分かります」という共感が基本

**語調**:
- 極めてフレンドリーで親しみやすい
- 友人に語りかけるような温かさ
- 上から目線は絶対にNG
- 読者と同じ目線、むしろ少し下から支える立場

## 記事構成のテンプレート

### タイトル
```
# 【〜が解説/実体験/インタビュー】〜【結論：〜】
```

**特徴**:
- 【】で囲んで注目を引く
- 「元公務員が解説」など信頼性を示す
- 【結論：〜】で答えを先に提示することも多い
- 日付を2行目に記載（例：2024/1/04）

### はじめにセクション

**必須要素**:
1. よくある誤解を提示
```markdown
**誤った認識：**
- 〜すればいい
- 〜は簡単
- 〜だけで大丈夫
```

2. 正しい認識を示す
```markdown
**正しい認識：**
〜という考え方が大切です！実は〜なんですよ。
```

3. 問題提起と記事の価値提示

4. 筆者の紹介
```markdown
### この記事を書いている人
- 元消防士（11年勤めたのち、退職）
- 現役ITエンジニア
- 〜の経験有り
```

### 目次
- 必ず設置する
- 読者が全体像を把握できるように
- リンク形式で各セクションに飛べるようにする

### 本文セクション

**セクション構成**:
```markdown
## ポイント1：〜 / アドバイス1：〜

**誤った認識：**
〜すればいい、〜だけで大丈夫

**正しい認識：**
実は〜が大切なんです！〜という方法がおすすめですよ。

本文...

---
```

**重要**:
- 各セクション間は `---` で区切る
- 「誤った認識」→「正しい認識」の流れを使用
- 太字（**）で認識を強調
- 親しみやすい口調を維持（「〜ですよ」「〜なんです」）

### まとめセクション

**必須要素**:
```markdown
## まとめ：〜

本文...

後悔のない選択をしてください。
全力で応援しています。
```

## 語尾と表現のルール

### 基本的な語尾

**多用する語尾**:
- 「〜でしょう」（推測・提案）
- 「〜ですよ」（アドバイス・情報提供）
- 「〜ね」（共感・確認）
- 「〜よ」（強調）
- 「〜ましょう」（提案）
- 「〜ください」（お願い）

**避ける語尾**:
- 「〜である」（堅すぎる）
- 「〜だ」（冷たく感じる）
- 「〜すべき」（押し付けがましい）

### 感嘆符（！）の使用

**積極的に使用する場面**:
- 共感を示すとき：「本っ当によく分かります！」
- 励ますとき：「大丈夫です！」
- 強調するとき：「問題ナシ！」
- 前向きなメッセージ：「チャンスは必ず来ます！」

**使用頻度**: 1段落に1〜2回程度

### 口語的表現

**積極的に使用**:
- 「〜って」：「転職活動ってハードル高そうに感じますけど」
- 「〜かな」：「何があるかな？」
- 「〜けど」：「分かりますけど」
- 「〜じゃない」：「難しいことじゃないけど」
- 「ちょっと」：「ちょっと落ち着いてください（笑）」
- 「本っ当に」：強調のために「っ」を入れる

### 共感表現の定型句

**必須表現**:
- 「その気持ち、本っ当によく分かります！」
- 「考えてしまう気持ち、本当によく分かります。」
- 「大丈夫です！」
- 「問題ありません！」
- 「それでも〜」（逆接での励まし）

## 文章の長さとリズム

### 段落の長さ
- **1段落：1〜3文**が基本
- 長くても5文まで
- 視覚的な読みやすさを重視

### 文の長さ
- **短文を基本**とする
- 1文は30〜50文字程度
- 長くなる場合は、箇条書きや改行で分割

### リスト・箇条書きの活用

**頻繁に使用**:
```markdown
- 項目1
- 項目2
- 項目3
```

**ネストも活用**:
```markdown
### 理由1：〜

説明文...

### 理由2：〜

説明文...
```

## テーマ別の語り口

### 不安を和らげるとき

```markdown
**誤った認識：**
〜は難しそうで不安

**正しい認識：**
大丈夫です！〜という方法があるんですよ。

私自身も〜で不安でしたが、〜したら〜できました。
```

### 厳しい現実を伝えるとき

```markdown

〜すれば簡単にできる

**正しい認識：**
正直に言うと、〜は簡単ではありません。

ただし、〜すれば道は開けます！私自身、〜という経験をしましたが、諦めなければ必ずチャンスは来ます。
```

### 励ますとき

```markdown
**誤った認識：**
もう遅すぎる、今からじゃ無理

**正しい認識：**
あなたにとって、今日が一番若い日ですから！

〜は大変ですが、それでも〜する価値は十分にありますよ。
```

## 禁止事項

### やってはいけない表現

❌ **上から目線**
- 「〜すべきだ」
- 「〜しなければならない」
- 「当然〜」

❌ **冷たい・突き放す表現**
- 「それは無理です」
- 「諦めてください」
- 「〜するしかない」

❌ **堅苦しい表現**
- 「である調」
- 専門用語の多用（説明なしで）
- 「貴殿」「貴方様」などの過剰な敬語

❌ **無責任な楽観**
- 「簡単です！誰でもできます！」
- 根拠のない「絶対大丈夫」

### やるべきバランス

✅ **現実的 + 希望を持てる**
```markdown
30代での未経験転職は、正直簡単ではありません。

ただし、〜という準備をすれば、チャンスは十分にあります。

私自身、34社落ちましたが、諦めずに続けて内定を得ることができました。
```

## NGワード・フレーズ

### 使ってはいけない表現

❌ 「〜すべきである」
❌ 「〜せざるを得ない」
❌ 「当然ながら」
❌ 「言うまでもなく」
❌ 「〜に違いない」
❌ 「絶対に〜」（根拠がない場合）
❌ 「誰でも簡単に」
❌ 「必ず成功する」

### 推奨する言い換え

✅ 「〜すべき」→「〜することをおすすめします」「〜した方がいいですよ」
✅ 「無理」→「簡単ではありません」「ハードルは高いです」
✅ 「ダメ」→「おすすめしません」「避けた方がいいです」
✅ 「絶対」→「ほぼ間違いなく」「高い確率で」

## 記事タイプ別のアプローチ

### タイプ1：アドバイス記事

**特徴**:
- 「〜の方法」「〜するには」
- アドバイス1、2、3...と番号付け
- 実体験ベースの具体的な提案

**構成**:
```markdown
# 【元〜が解説】〜の方法【結論：〜】

## はじめに
[読者の悩み + 共感]

## アドバイス1：〜
## アドバイス2：〜
## アドバイス3：〜

## まとめ
```

### タイプ2：インタビュー記事

**特徴**:
- 第三者の声を紹介
- 質問1、2、3...の形式
- より具体的な情報提供

**構成**:
```markdown
# 【実体験インタビュー】〜

## はじめに
## プロフィール
## 質問1：〜
## 質問2：〜
## まとめ
```

### タイプ3：検証・考察記事

**特徴**:
- 「〜は可能か？」という問い
- データや事実を提示
- 【結論：〜】で答えを明示

**構成**:
```markdown
# 〜は可能？【結論：〜】

## はじめに
## 検証ポイント1
## 検証ポイント2
## まとめ：結論
```

## 遊之介の文体の本質

### 核心的な価値観

1. **読者ファースト**
   - 読者の気持ちを最優先
   - 不安を和らげ、希望を持たせる
   - でも現実も正直に伝える

2. **実体験の重視**
   - 机上の空論ではない
   - 自分が経験したことを語る
   - 失敗も含めて正直に開示

3. **行動を促す**
   - 読んで終わりではなく
   - 具体的な次の一歩を示す
   - ハードルを下げて背中を押す

4. **温かさと誠実さ**
   - 友人に語りかけるような温かさ
   - でも嘘はつかない誠実さ
   - 両方のバランスが大事

### この文体で大切にしていること

**読者の「誰か」を具体的に想像する**
- 30代の公務員
- 転職に悩んでいる
- 家族がいて、簡単には辞められない
- でも、このままでいいのか疑問を感じている

**その人に向けて、友人として語る**
- 「分かるよ、その気持ち」
- 「俺も同じだったから」
- 「大丈夫、こうすればいいよ」
- 「一緒に頑張ろう」

**希望と現実のバランス**
- 「簡単じゃないよ、正直」
- 「でも、できないわけじゃない」
- 「俺もできたんだから」
- 「諦めなければ道は開ける」

## 文体習得のための練習方法

### 練習1：書き換え練習

堅い文章を、新しいスタイルに書き換える

**元の文**:
```
転職には準備が必要である。特にスキルの習得は重要だ。
```

**書き換え後**:
```
**誤った認識：**
転職は準備なしでもできる

**正しい認識：**
まずはスキルを身につけることから始めましょう！

準備と言っても、そんなに難しく考えなくて大丈夫ですよ。
```

### 練習2：認識の提示形式の作成

テーマを決めて、誤った認識と正しい認識を作ってみる

**テーマ**: 「30代での転職は遅いか？」

```markdown
**誤った認識：**
もう30代だし、転職は手遅れ

**正しい認識：**
そんなことありません！あなたにとって、今日が一番若い日ですから。

確かに20代に比べると、30代の転職はハードルが高いです。

でも、不可能ではありません。

私自身、34歳で転職しましたからね！
```

### 練習3：実体験の語り方

自分の経験を、親しみやすいスタイルで語ってみる

```markdown
**誤った認識：**
落ち続けたら、もう諦めるしかない

**正しい認識：**
諦めなければ、必ずチャンスは来ます！

私自身、転職活動中は本当に辛かったです。

34社も応募して、そのほとんどが書類選考で落とされました。

> 「自分には価値がないのか...」

そんな風に思ったこともありました。

でも、諦めずに続けたら、最終的に内定を得ることができたんです。
```

## まとめ：この文体のエッセンス

この文体の本質は、**読者の誤解を正し、希望を与え、行動を促す**ことです。

### 忘れてはいけない3つのポイント

1. **誤った認識を明確にする**
   - よくある誤解・思い込みをはっきり示す

2. **正しい認識を親しみやすく伝える**
   - 「実は〜なんです！」「〜がおすすめですよ」

3. **実体験と励ましで信頼を得る**
   - 「私もそうでした」「大丈夫、あなたならできます」

### 最も大切なこと

**読者の人生を変える一押しになる記事を書く**

それが、遊之介スタイルの記事の使命です。
